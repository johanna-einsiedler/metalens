3Describe your task▾

Configure your MASEMiner extraction

Which effect-size statistics should the model extract? Use short codes the literature recognises (r Pearson, or odds ratios, smd standardised mean differences…). The right of the colon is the label shown in the prompt.
Canonical variables/constructs (the output's var1/var2). Short code + a definition that helps the model map paper-specific operationalisations. Add a third :: chunk for synonyms.
Paste the verbatim item texts in order (1: item text) so the model matches reordered/paraphrased items. Leave empty for copyrighted instruments.
